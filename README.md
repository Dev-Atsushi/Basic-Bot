# 🥥 | How to configure?
Put your token in "TOKEN"
- config.js
```js
module.exports = {
  prefix: "YOUR_PREFIX",
  token: "YOUR_TOKEN"
}
```
Put your prefix in "YOUR_PREFIXO"
Put your token in "YOUR_TOKEN"
